# URL Shortener (FastAPI + Redis)

A scalable URL Shortener backend built using FastAPI with Redis caching and SQLite storage.

---

## Features
- Shorten long URLs
- Fast redirection using Redis cache
- Persistent storage using SQLite
- Docker-based Redis setup
- RESTful API design

---

## Tech Stack
- Python
- FastAPI
- Redis
- SQLite
- Docker
